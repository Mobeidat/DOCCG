== 简介 ==

DOCCG是由TingBiao Wang开发的WordPress主题。 您可以将其用于商业和个人项目。

== 安装 ==

1.在管理面板中，转到外观->主题，然后单击"添加新"按钮。
2.在搜索表单中键入DOCCG，然后按键盘上的" Enter"键。
3.单击"激活"按钮立即使用您的新主题。
4.在管理面板中导航至"外观">"自定义"，然后自定义口味。
